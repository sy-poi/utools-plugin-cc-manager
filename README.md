# Claude Code 会话管理器

一款 [uTools](https://www.u-tools.cn/) 插件，用于浏览、管理和查看本地 Claude Code 的会话记录。

> 目前仅测试了Windows端，其他端用户如果有不好使的功能请反馈，我会尽快修复。

## 功能特性

### 会话浏览

- 自动扫描 `~/.claude/projects` 目录下的所有项目和会话文件
- **懒加载**：启动时快速显示项目列表，展开项目时才加载完整会话详情，减少首次加载耗时
- 项目按实际工作目录路径显示，长路径自动缩短，hover 显示完整路径
- 会话按最后消息时间倒序排列，最新活跃的项目和会话排在最前
- 自动识别会话名称：优先读取自定义标题，否则取最后一条有效用户消息
- 智能跳过无效消息（工具响应、命令注入、系统注入等）作为会话名，支持多 block 消息中提取真实用户文本
- **收藏/置顶**：星标重要会话，收藏的会话在项目内置顶显示
- **子对话（Subagent）**：展开会话可查看其下的子对话列表，点击进入子对话视图

### 会话查看

- 双栏布局：左侧项目/会话列表，右侧会话内容查看
- **Markdown 渲染**：AI 回复使用 markdown-it 渲染，支持代码块、表格、列表、链接、标题等
- **代码块复制**：每个代码块右上角显示语言标签和复制按钮，一键复制代码
- 智能消息合并：连续的 AI 消息自动合并，tool_result、sourceToolUseID/sourceToolAssistantUUID 工具响应合并到对应的 AI 消息中
- **命令注入合并**：isMeta 命令注入消息（如 /init 的提示词）合并到命令消息中，显示为可折叠的 Prompt 块
- **工具调用卡片**：Read/Write/Edit/Bash 等工具调用以卡片形式展示，折叠/展开
- **Edit Diff 视图**：Edit 工具以红绿 diff 形式展示代码变更，支持行内差异高亮
- **图片预览**：用户上传的图片可点击放大、复制到剪贴板、另存为文件
- **XML 标签解析**：stdout/stderr/命令/IDE 事件等 XML 标签结构化显示
- **系统事件展示**：用户中断以分隔线样式展示，系统通知以橙色气泡卡片展示，用户命令（含 command-name）正常显示
- **上下文压缩摘要**：压缩后的会话显示"上下文压缩"标记块，可展开查看摘要内容；仅含摘要的会话显示恢复引导提示
- **API 错误展示**：API 错误消息以红色错误样式显示
- **Token 统计**：每条 AI 消息底部显示模型、token 用量、缓存命中、耗时、停止原因
- **会话内搜索**：支持关键词高亮定位、上下导航，支持区分大小写、全字匹配、正则表达式
- **搜索快捷键**：Ctrl+F 打开搜索、F3/Shift+F3 上下导航、Esc 关闭、自动聚焦输入框
- 兼容标准 JSONL 和美化多行 JSON 两种会话文件格式
- 实时监听：选中会话后自动监听文件变化，新消息实时刷新
- 自动滚动：文件更新时若滚动条在底部则自动跟随，否则保持当前位置

### 会话操作

- **搜索过滤**：按会话名、sessionId、工作目录模糊搜索，搜索时自动展开匹配项目
- **侧边栏右键菜单**：右键点击会话或点击"⋮"按钮弹出菜单，包含新窗口打开、终端恢复、收藏、重命名、删除（Ctrl 直接删除跳过确认）
- **新窗口打开**：侧边栏和会话视图均可将任意会话（含子对话）在独立窗口中查看
- **重命名会话**：标题栏编辑按钮或侧边栏右键菜单
- **收藏会话**：标题栏星标按钮或侧边栏右键菜单
- **删除会话**：更多菜单中删除，需二次确认；侧边栏右键菜单删除支持 Ctrl 跳过确认
- **批量删除**：多选模式下勾选多个会话，一键批量删除
- **Fork 分支**：从任意 AI 消息处截断，命名并创建新的会话分支；也可以上下文压缩摘要为起点创建分支；Fork 成功后提示是否立即在终端中打开
- **恢复会话**：底部恢复按钮栏或点击终端图标，在新终端窗口中执行 `claude --resume {sessionId}`，支持 Windows / macOS / Linux
- **复制恢复命令**：一键复制 `claude --resume {sessionId}` 到剪贴板
- **打开源文件目录**：在更多菜单中打开会话 JSONL 源文件所在目录
- **打开项目目录**：项目文件夹悬浮显示打开按钮，点击在文件管理器中打开
- **复制消息**：悬浮在消息上显示复制按钮，仅复制纯文本内容（排除工具调用等）
- **快速滚动**：右下角顶部/底部快速滚动按钮
- **更多菜单**：标题栏右侧 ⋮ 下拉菜单，包含打开目录、分享（二级下拉）、删除

### 导出

- **导出 Markdown**：会话导出为 `.md` 文件
- **导出 HTML**：基于 DOM 克隆导出为独立 HTML 文件，样式与预览完全一致，支持点击展开/收起工具调用和思考过程
- **导出长图**：基于 DOM 克隆导出为 PNG 长图，样式与预览完全一致
- **导出选项**：可自定义显示 Session ID、项目路径、统计信息、工具详情、思考过程，工具文件路径支持缩略/完整/隐藏

### 界面交互

- 明暗主题切换（跟随系统 / 手动切换）
- 侧边栏可收起/展开，滑动动画
- 右侧抽屉式设置面板

### 设置

- **终端命令**：自定义恢复会话使用的命令，默认 `claude`
- **终端程序**：自动识别或自定义命令模板，自定义模式支持 `${cwd}`（项目目录）、`${cc_cmd}`（恢复命令）占位符，例：`start cmd /c "cd ${cwd} && ${cc_cmd}"`
- **主题模式**：系统跟随 / 亮色 / 暗色

## 技术栈

> 本项目完全由 Claude Code 生成

- **Vue 3** + Composition API
- **Vite 6** 构建
- **markdown-it** Markdown 渲染
- **html2canvas** 长图导出
- **纯 CSS** 手写样式，无第三方 UI 库
- **Node.js** preload 层处理文件系统操作
- **uTools API** 插件平台集成与数据持久化

## 安装使用

1. 安装依赖：`pnpm install`
2. 开发模式：`pnpm run dev`
3. 构建：`pnpm run build`
4. 在 uTools 开发者工具中加载 `public/plugin.json` 或 `dist` 目录
5. 在 uTools 中输入「CC会话管理」启动插件
