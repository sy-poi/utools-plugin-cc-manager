# Changelog

## v1.0.0

1. 新增记忆管理：项目目录下有 `memory/MEMORY.md` 时，会话列表显示"记忆管理"入口，展示记忆总大小和文件数
2. 记忆文件非编辑状态下渲染为 Markdown，YAML frontmatter 以样式化 key-value 块展示
3. 记忆管理页支持在线编辑和保存（有改动时保存按钮高亮），Ctrl+S 批量保存
4. MEMORY.md 中引用的文件链接可点击页内导航，滚动停止后目标卡片高亮闪烁两次
5. MEMORY.md 列表项前显示管理按钮（默认小点，悬浮变齿轮），可删除记忆并同步移除引用行
6. 未被 MEMORY.md 引用的 `.md` 文件标记为"无效记忆"并显示删除按钮
7. 支持一键清空所有记忆（二次确认），记忆变更后侧边栏大小/数量实时同步
8. 记忆文件被外部修改时自动静默更新；若本地有未保存改动则弹出冲突对话框
9. 冲突对话框以左右对齐的 side-by-side diff 展示差异，配对行高亮字符级变更，支持复制任一侧内容
10. 侧边栏新增过滤按钮，可筛选"仅展示有记忆的项目"
11. 项目目录支持右键菜单，新增"新建会话"和"打开目录"两项

## v0.5.0

1. 项目懒加载：启动时快速显示项目列表，展开时才加载会话详情，减少首次加载耗时
2. 子对话（Subagent）支持：侧边栏可展开查看会话的子对话列表，工具调用卡片新增"查看"链接直接打开子对话
3. 新窗口打开会话：会话视图和侧边栏均可在独立窗口中打开任意会话或子对话
4. 侧边栏右键菜单：取代原操作按钮，支持新窗口打开、终端恢复、收藏、重命名、删除（Ctrl 直接删除）
5. 上下文压缩摘要展示：识别并渲染压缩摘要块，仅含压缩摘要的会话显示引导恢复提示，大纲侧边栏中包含压缩节点
6. 以压缩摘要创建 Fork 分支，Fork 成功后弹出快速恢复确认卡
7. 会话底部新增恢复会话按钮栏
8. 重命名/Fork 会话时同步更新 agent-name 字段
9. 时间格式新增秒数显示
10. Snackbar 消失时间从 3s 缩短为 2.5s，鼠标悬停时暂停消失，离开后延迟 1s 再消失
11. 新增「终端程序」设置：自动识别或自定义命令模板，支持 `${cwd}`（项目目录）、`${cc_cmd}`（恢复命令）占位符

## v0.4.0

1. Bash 工具展开后以代码块的方式显示命令
2. 工具调用展开后不再截断内容，改为超高内容显示滚动条
3. 过滤空的 Thinking 块，不再渲染无内容的思考过程
4. AI 消息底部 token 数量格式化为 K/M/B 单位，调用时长格式化为 xxhxxmxxs 形式
5. 新增消息大纲侧边栏，点击头部列表图标展开，可快速跳转到任意用户消息

## v0.3.0

1. 重构导出架构：图片和 HTML 导出共用 DOM 克隆方案，彻底消除两套样式代码
2. 导出的 HTML 支持点击展开/收起工具调用和思考过程
3. 导出选项新增工具文件路径显示模式（缩略/完整/隐藏）
4. 会话标题栏新增收藏标星、重命名按钮
5. 右侧新增"更多"下拉菜单，包含打开源文件目录、分享（二级下拉）、删除会话
6. 标题栏新增复制恢复命令按钮
7. 搜索栏支持自动聚焦、Ctrl+F/F3/Shift+F3/Esc 快捷键
8. 搜索栏新增区分大小写、全字匹配、正则匹配选项
9. API 错误消息（isApiErrorMessage）显示为红色错误样式
10. 带 sourceToolUseID/sourceToolAssistantUUID 的用户消息正确合并为工具调用结果
11. 命令注入消息（isMeta）合并到命令消息中，显示为可折叠的 Prompt 块
12. 含 command-name 的用户命令消息不再被误判为系统事件
13. 修复会话名称提取：跳过工具响应和命令注入消息，支持多 block 跳过系统注入取真实用户文本
14. 修复搜索时匹配文本重复显示的问题
15. 搜索高亮当前项去掉边框，仅用背景色区分

## v0.2.0

1. 修复了 Fork 截断位置不正确的问题，现在能准确截取到完整的 AI 回复
2. 修复了真实用户消息被错误合并到 AI 消息中的问题
3. 新增系统事件独立展示：中断消息显示为分隔线，通知类消息显示为橙色气泡卡片
4. 优化了系统注入消息的识别逻辑，提取共用函数统一判定

## v0.1.0

首个发布版本，目前支持以下功能：

1. 支持自动扫描 ~/.claude/projects/ 下所有项目和会话，按最近活跃时间排序
2. 支持 AI 回复 Markdown 渲染，代码块带语言标签和一键复制
3. 支持智能消息合并（连续 AI 消息、工具调用与结果自动配对）
4. 支持 Edit 工具 Diff 视图（红绿高亮 + 行内差异标注）
5. 支持工具调用卡片化展示（Read/Write/Edit/Bash/Grep 等）
6. 支持 Token 用量统计（模型、输入/输出 tokens、缓存命中、耗时）
7. 支持图片预览、复制到剪贴板、另存为文件
8. 支持会话内关键词搜索高亮与上下导航
9. 支持会话重命名、删除（Ctrl 跳过确认）、批量删除
10. 支持 Fork 分支：从任意 AI 消息处截断创建新会话
11. 支持收藏/置顶重要会话
12. 支持在终端中恢复会话（claude --resume）
13. 支持导出为 Markdown / HTML / 长图，可自定义导出内容
14. 支持明暗主题切换（跟随系统 / 手动）
15. 支持实时监听会话文件变化，自动刷新
16. 兼容 Windows / macOS / Linux 跨平台终端调用
