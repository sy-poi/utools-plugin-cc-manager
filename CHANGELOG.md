# Changelog

## v0.1.0

首个发布版本，目前支持以下功能：

1. 支持自动扫描 ~/.claude/projects/ 下所有项目和会话，按最近活跃时间排序
2. 支持 AI 回复 Markdown 渲染，代码块带语言标签和一键复制
3. 支持智能消息合并（连续 AI 消息、工具调用与结果自动配对）
4. 支持 Edit 工具 Diff 视图（红绿高亮 + 行内差异标注）
5. 支持工具调用卡片化展示（Read/Write/Edit/Bash/Grep 等）
6. 支持 Token 用量统计（模型、输入/输出 tokens、缓存命中、耗时）
7. 支持图片预览、复制到剪贴板、另存为文件
8. 支持会话内关键词搜索高亮与上下导航
9. 支持会话重命名、删除（Ctrl 跳过确认）、批量删除
10. 支持 Fork 分支：从任意 AI 消息处截断创建新会话
11. 支持收藏/置顶重要会话
12. 支持在终端中恢复会话（claude --resume）
13. 支持导出为 Markdown / HTML / 长图，可自定义导出内容
14. 支持明暗主题切换（跟随系统 / 手动）
15. 支持实时监听会话文件变化，自动刷新
16. 兼容 Windows / macOS / Linux 跨平台终端调用
